1. inflation /ɪnˈfleɪʃn/
n. 通货膨胀
词根词缀拆解
in-：进入，使…
flat：= blow 吹、膨胀（来自拉丁语 flare 吹）
-ion：名词后缀
原意来源：原本指 “吹气使膨胀”，经济领域引申为货币膨胀、物价上涨
记忆结构：in (进入)+flat (吹膨)+ion → 不断吹大 → 通胀
2. eurozone /ˈjʊərəʊzəʊn/
n. 欧元区
词根词缀拆解
euro：欧元（Europe 缩写 / 变体）
zone：区域、地带
记忆结构：euro (欧元)+zone (区域) → 使用欧元的区域
3. previously /ˈpriːviəsli/
adv. 先前，此前
词根词缀拆解
pre-：在前，提前
vi/vis：= see 看（video/visible 同根）
-ous：形容词后缀 → previous
-ly：副词后缀
原意来源：“在眼前看到之前的” → 先前的
记忆结构：pre (前)+vis (看)+ious+ly → 之前看到过地 → 此前
4. energy /ˈenədʒi/
n. 能源；精力
词根词缀拆解
en-：使…
erg：= work 工作、做功（ergonomics 工效学 同根）
-y：名词后缀
原意来源：古希腊语 energeia 活动、运作 → 能做功的力量
记忆结构：en (使)+erg (做功)+y → 能做功的东西 → 能源 / 精力
5. pronounced /prəˈnaʊnst/
adj. 明显的，显著的
词根词缀拆解
pro-：向前，公开
nounce/nunci：= speak 说、宣布（announce 宣布 同根）
-ed：形容词化
原意来源：本义 “被清楚说出的” → 引申为非常明显、一眼能看出来的
记忆结构：pro (公开)+nounce (说)+ed → 被明说出来的 → 明显的
6. consumer /kənˈsjuːmə(r)/
n. 消费者
词根词缀拆解
con-：加强语气
sume/sumpt：= take 拿、取、消耗（assume 承担 同根）
-er：表人
原意来源：“拿走、消耗物品的人”
记忆结构：con+sume (消耗)+er → 消耗商品的人 → 消费者
7. provisional /prəˈvɪʒənl/
adj. 临时的，暂定的
词根词缀拆解
pro-：向前，提前
vis：= see 看 → 预见、准备
-ion：名词 → provision 预备、供应
-al：形容词
原意来源：提前准备好的、应急用的 → 临时的
记忆结构：pro (前)+vis (看)+ion+al → 预先准备的 → 暂定 / 临时的
8. estimate /ˈestɪmət/
n. 估计，估值
词根词缀拆解
estim：= value 价值、判断（拉丁语 aestimare 估价）
-ate：名词 / 动词后缀
原意来源：对价值进行判断 → 估算
记忆结构：estim (估价)+ate → 估价结果 → 估计值
9. revision /rɪˈvɪʒn/
n. 修正，修订
词根词缀拆解
re-：再，重新
vis：= see 看 → 再看一遍
-ion：名词后缀
原意来源：重新查看并修改 → 修订
记忆结构：re (再)+vis (看)+ion → 再看一遍并改 → 修正

## 一、核心栏目/政治经济类词汇
### 1. **politics**
- 拆解：**polit-** + **-ics**
- 词根：**polit = city / citizen**（城邦、公民，来自希腊语 *polis*）
- 后缀：**-ics** 表学科/活动
- 原意：城邦治理、公民事务
- 记忆：公民→城邦事务→**政治**

### 2. **economy**
- 拆解：**eco-** + **-nomy**
- 词根：**eco = house/home**（家/管理）；**nomy = law/management**（规则、管理）
- 原意：家庭管理 → 扩展为国家资源管理
- 记忆：管家→管国家资源→**经济**

### 3. **finance**
- 拆解：**fin-** + **-ance**
- 词根：**fin = end / boundary / pay off**（结束、了结债务）
- 后缀：**-ance** 名词
- 原意：了结债务、支付
- 记忆：把账了结→**金融、财务**

### 4. **accreditation**
- 拆解：**ac-** + **cred-** + **-it-** + **-ation**
- 词根：**cred = believe/trust**（相信）
- 前缀：**ac- = to**
- 原意：给予信任、认可
- 记忆：相信→授予资格→**认证、资质认定**

### 5. **monopoly**
- 拆解：**mono-** + **-poly**
- 词根：**mono = one**（单一）；**poly = sell**（售卖）
- 原意：单独售卖
- 记忆：一家独卖→**垄断**

### 6. **diversity**
- 拆解：**di-** + **vers-** + **-ity**
- 词根：**vers = turn**（转、转向不同方向）
- 原意：转向不同 → 多种多样
- 记忆：转不同方向→**多样性**

### 7. **equity**
- 拆解：**equ-** + **-ity**
- 词根：**equ = equal/even**（平等、平）
- 原意：平等、公正
- 记忆：平等→**公平；资产净值**

### 8. **inclusion**
- 拆解：**in-** + **clus-** + **-ion**
- 词根：**clus = close/shut**（关、包含）
- 原意：关入、包含在内
- 记忆：包含进来→**包容、包容性**

## 二、社会/教育/法律类
### 9. **education**
- 拆解：**e-** + **duc-** + **-ation**
- 词根：**duc = lead**（引导）
- 前缀：**e- = out**
- 原意：引导出来（天赋/潜能）
- 记忆：把潜能引出来→**教育**

### 10. **prerequisite**
- 拆解：**pre-** + **re-** + **quis-** + **-ite**
- 词根：**quis = seek/search**（寻求、要求）
- 前缀：**pre = before**
- 原意：事先要求的东西
- 记忆：提前要求→**先决条件、前提**

### 11. **reform**
- 拆解：**re-** + **form**
- 词根：**form = shape/form**（形状、形式）
- 前缀：**re = again/back**
- 原意：重新塑造
- 记忆：重新塑形→**改革**

### 12. **corruption**
- 拆解：**cor-** + **rupt-** + **-ion**
- 词根：**rupt = break**（断裂、破坏）
- 前缀：**cor- = completely**
- 原意：彻底破坏（道德/制度）
- 记忆：彻底崩坏→**腐败**

### 13. **redistricting**
- 拆解：**re-** + **dis-** + **tric-** + **-ing**
- 词根：**tric = district/area**（区域）
- 原意：重新划分区域
- 记忆：重划选区→**选区重划**

## 三、媒体/观点/表达类
### 14. **expression**
- 拆解：**ex-** + **press-** + **-ion**
- 词根：**press = press/push**（压、挤）
- 前缀：**ex = out**
- 原意：挤压出来→说出/表现
- 记忆：把想法挤出来→**表达；言论**

### 15. **opinion**
- 拆解：**op-** + **-ion**
- 词根：**op = choose/think**（选择、思考）
- 原意：经过思考得出的判断
- 记忆：思考后的看法→**观点、舆论**

### 16. **editorial**
- 拆解：**edit-** + **-or-** + **-ial**
- 词根：**edit = bring up/arrange**（整理、出版）
- 后缀：**-ial** 形容词/名词
- 原意：编辑整理的内容
- 记忆：编辑部文章→**社论**

### 17. **document**
- 拆解：**doc-** + **-u-** + **-ment**
- 词根：**doc = teach/prove**（教、证明）
- 原意：用来证明/记录的东西
- 记忆：证明记录→**文件；记录**

## 四、高频动词/抽象名词
### 18. **mandate**
- 拆解：**man-** + **-date**
- 词根：**man = hand**；**date = give**（交给、托付）
- 原意：亲手交付命令
- 记忆：下令要求→**授权；强制要求**

### 19. **spreads**
- 拆解：**spread**（原始日耳曼语）
- 原意：伸开、铺开
- 记忆：铺开传播→**传播；推行**

### 20. **subscribe**
- 拆解：**sub-** + **scrib-** + **-e**
- 词根：**scrib = write**（写）
- 前缀：**sub = under**
- 原意：写下名字（在下方签名）
- 记忆：签名加入→**订阅；同意**

### 21. **presidential**
- 拆解：**preside-** + **-nt-** + **-ial**
- 词根：**sid = sit**（坐）
- 前缀：**pre = before**
- 原意：坐在前面主持
- 记忆：主持国家→**总统的**
